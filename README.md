# Payroll Management System

A desktop-based Payroll Management System developed using C# and .NET with SQL Server for database management.

## 📌 Project Description

This application allows organizations to manage employees, bonuses, salary payments, attendance, and advance payments in a structured way.

The system provides an administrative dashboard to monitor salary and bonus records and manage employee-related data efficiently.

## 🚀 Features

- Add, update, delete employees
- Manage bonuses and advance payments
- Record attendance
- Issue salaries
- Dashboard overview with salary and bonus summaries
- SQL Server database integration

## 🛠 Technologies Used

- C#
- .NET Framework / .NET
- Windows Forms
- SQL Server
- ADO.NET / Entity Framework
